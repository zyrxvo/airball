# Examples in AIRBALL
Examples are the easiest way to learn how to use AIRBALL. This page contains a list of all examples that come with AIRBALL.

!!! Info
    You can find the source code for all examples in the `docs/examples/` directory.

## Available Examples
- [Getting Started](getting-started.ipynb)
- [Basic flyby setups](basics.ipynb)
- [Initial Mass Functions (IMFs)](imf.ipynb)
- [Sampling the effects of different stellar environments](stellar-environments.ipynb)
- [Comparing the differences between successive and concurrent flybys](multiple-flybys.ipynb)
- [Adiabatic tests to verify the flyby results are accurate](adiabatic-tests.ipynb)
